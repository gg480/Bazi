AI 编程提示词 · 模块一：排盘核心算法

你是一位精通中国传统命理学的全栈工程师。请用 TypeScript 实现一个完整的八字排盘核心算法库，要求： 【数据结构】 - 定义 BaziChart 接口，包含：pillars（四柱干支）、wuxing（五行计数）、shishen（十神分析）、riZhuStrength（日主强弱评分0-100） - 定义 DaYun 接口，包含每步大运的干支、起止年龄、流年数组 【算法要求】 1. 公历转农历：使用 lunar-javascript 库，节气换月以精确到分钟的节气时刻为准 2. 四柱推算： - 年柱：以立春为界（非正月初一） - 月柱：以12节气为界，提供节气精确时刻对照表 - 日柱：使用干支纪日算法（甲子日为基准日） - 时柱：23:00-01:00 为子时，含跨日判断 3. 十神推算：以日干为我，实现完整的生克制化十神映射表（正官、七杀、正印、偏印、比肩、劫财、食神、伤官、正财、偏财） 4. 日主强弱：实现月令得分（旺相休囚死）+ 地支藏干透出得分算法，输出强/中/弱三级判断 5. 大运推算：实现阳男阴女顺排、阴男阳女逆排规则；起运岁数 = 出生到最近节气天数 / 3 【输出】 - 导出 calculateBazi(birthDate: Date, birthHour: number, gender: 'male'|'female', birthplace?: {lat:number,lng:number}) 函数 - 包含完整 JSDoc 注释和边界情况处理（如子时跨日、节气当天出生等） - 附带单元测试用例：使用3个真实历史名人八字验证准确性

AI 编程提示词 · 模块二：AI 解读报告生成

请实现一个 Next.js API Route：/api/report/generate，功能是将排盘结果转化为 AI 八字解读报告。 【输入接口】 接收 POST 请求，body 为序列化后的 BaziChart 对象（含四柱、五行、十神、大运信息） 【Prompt 工程要求】 System Prompt 需包含： 1. 角色设定：你是一位通晓子平真诠、滴天髓、穷通宝鉴的资深命理师，语言通俗易懂，不用晦涩术语 2. 输出框架（严格 JSON 结构）： { "personality": "性格特质（200字）", "career": "事业财运分析（200字）", "relationship": "感情婚姻分析（200字）", "health": "健康提示（150字）", "wuxing_advice": "五行调候建议（含颜色/方位/玉石推荐）（150字）", "dayun_summary": "当前大运解读（150字）", "overall_score": 1-100的综合运势评分 } 3. 安全引导：所有解读以积极引导为主，规避"此命克夫/克妻/短命"等负面绝对化表述，改用"需注意XX方面的压力"替代 4. 数据注入模板：将 BaziChart 数据格式化为自然语言描述注入 User Message 【工程实现】 - 使用 Anthropic SDK + streaming 响应（Server-Sent Events） - 免费用户：claude-haiku-4-5 + max_tokens:800 - 付费用户：claude-sonnet-4-6 + max_tokens:2000 - 相同八字+大运的报告缓存到 Redis，TTL 30天 - 实现请求限流：免费用户每日最多3次生成
